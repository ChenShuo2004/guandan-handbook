# 掼蛋知识库

一份按学习路径组织的掼蛋中文知识库。

这里不收集零散口诀，也不把某一种牌桌经验写成唯一答案。我们从规则开始，逐步讨论组牌、记牌、牌权、炸弹、搭档配合和残局判断，并把每个主题写成可以阅读、练习和复盘的独立文件。

> 当前版本：V1。基础阅读路径、规则口径、实用方法、训练框架与资料来源已经形成完整闭环。

## 写作约定

本库会明确区分三类内容：

- **规则**：必须注明采用的版本；正式比赛以当次竞赛规程和补充规定为准。
- **常见经验**：说明适用条件，不使用“必打”“绝不能”等无条件结论。
- **推断与假设**：写出依据、仍然未知的部分，以及判断失败的代价。

每个章节用“资料来源”说明依据。系统内课程、截图 OCR 和视频拆解用于提供经验、牌例与问题线索；正式规则优先回到公开规则原文。

此外，本库不教授作弊、约定暗号或其他隐蔽信号；不根据一两局牌给人贴人格标签。

## 内容维护约定

新增牌例时，应尽量补全级数、座次、四家剩余张数、当前牌权、已知出牌和目标；只有结论、没有条件的口诀很难复用。发现规则口径、链接或牌例有问题时，应补充或更正来源，并说明哪一项条件会改变结论。

本库由既有知识材料重新整理而成，不是原知识库的镜像导出。材料对应关系与来源见[来源说明](references/sources.md)。
