# 规则版本说明

> 状态：持续复核。最后核对日期：2026-07-15。

掼蛋在朋友局、线上平台、地方比赛和正式赛事中可能采用不同规则。本库的正文只使用足以解释学习方法的共同框架，不把某一套细节写成所有牌桌都适用的唯一规则。

## 当前参考口径

### 《世界掼牌（掼蛋）竞赛规则（2025）》

- 文件：[查看 PDF](https://gawsf.b-cdn.net/files/%E4%B8%96%E7%95%8C%E6%8E%BC%E7%89%8C%EF%BC%88%E6%8E%BC%E8%9B%8B%EF%BC%89%E7%AB%9E%E8%B5%9B%E8%A7%84%E5%88%99%EF%BC%882025%EF%BC%89.pdf)
- 用途：当前规则正文的主要复核入口。
- 注意：它可以提供统一的竞赛语言，但不能自动替代具体比赛发布的竞赛规程、补充规定或平台规则。

### 国家体育总局棋牌运动管理中心文件

- [《掼牌（掼蛋）赛事办赛指南（试行）》](https://www.sport.gov.cn/qpzx/n27319064/n27319058/c27429584/content.html)
- [《掼牌（掼蛋）赛事纪律准则与处罚规定（试行）》](https://www.sport.gov.cn/qpzx/n5384/c25953388/content.html)

办赛指南明确，赛事文件通常包括竞赛规程、补充规定和补充通知。参加正式比赛时，应优先查看主办方当次发布的文件。

## 上桌前必须确认的差异

至少确认以下事项：

1. 出牌方向；
2. 连续牌型中 A 和级牌的使用方式；
3. 不同炸弹、同花顺和王炸的大小顺序；
4. 出完最后一手后的接风方式；
5. 升级幅度和过 A 条件；
6. 是否进贡、怎样抗贡、由谁首出；
7. 报牌、计时、判罚等赛场要求。

这些差异可能直接改变牌例答案。因此，本库新增牌例时应注明规则版本；无法确认时标记“待复核”，不使用绝对结论。

## 内容状态约定

- **已复核**：已经与标注版本的原始规则逐条核对；
- **待复核**：来源明确，但尚未完成逐条核对；
- **经验**：常见打法或教学方法，不是规则条文；
- **推断**：依据公开信息形成的局面判断，仍可能被新信息修正。

## 维护规则

发现版本冲突时，不直接覆盖旧结论。应记录：冲突发生在哪条规则、两个版本分别怎样规定、哪些正文或牌例受影响，以及本库接下来采用哪一个口径。
