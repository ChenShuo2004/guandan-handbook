# 规则来源与适用范围

掼蛋在朋友局、线上平台、地方比赛和正式赛事中可能采用不同规则。本库在具体页面列出资料来源、采用口径和适用范围，让读者可以直接回到原始依据。

## 本库的主要规则来源

### 《世界掼牌（掼蛋）竞赛规则（2025）》

- [查看 PDF](https://gawsf.b-cdn.net/files/%E4%B8%96%E7%95%8C%E6%8E%BC%E7%89%8C%EF%BC%88%E6%8E%BC%E8%9B%8B%EF%BC%89%E7%AB%9E%E8%B5%9B%E8%A7%84%E5%88%99%EF%BC%882025%EF%BC%89.pdf)
- 发布方：世界掼牌体育会。
- 本库用途：确定术语、牌型、牌点大小、出牌程序、接风、报牌、升级和贡还牌的基础口径。
- 相关位置：第三章“基本规则”，尤其是 PDF 第 5—8 页。

### 国家体育总局棋牌运动管理中心文件

- [《掼牌（掼蛋）赛事办赛指南（试行）》](https://www.sport.gov.cn/qpzx/n27319064/n27319058/c27429584/content.html)
- [《掼牌（掼蛋）赛事纪律准则与处罚规定（试行）》](https://www.sport.gov.cn/qpzx/n5384/c25953388/content.html)

本库主要用这些文件确认正式赛事的规则边界、竞赛文件关系和公平纪律要求。办赛指南指出，具体赛事还会发布竞赛规程、补充规定和补充通知，因此一份通用规则不能自动替代当次赛事文件。

## 系统内知识材料怎样使用

内部素材库 `Sources/内容平台与变现/掼蛋记牌训练器素材/` 同时包含完整课程、OCR 截图、视频拆解和二次分析。它们在本库中的职责不同：

| 材料 | 主要用途 | 不承担的职责 |
| --- | --- | --- |
| `截图线索-掼蛋教程.md` | 新手术语、牌型和流程的帮助说明 | 不单独决定正式规则 |
| `原文-第一课无逢配牌型组合.md` | 组牌、拆牌和减少轮次的课程经验 | 不作为规则条文 |
| `原文-第二课第三课逢人配与记牌残局.md` | 逢人配、关键牌记忆和残局案例 | 不把案例结论写成无条件定理 |
| `03-批量归类与V1能力树.md` | 确定记牌、炸弹、剩牌推理等学习模块 | 不证明具体打法一定正确 |
| `04-截图OCR与归类.md` | 区分规则、经验口诀和防作弊材料 | OCR 结果仍需回到原始来源核对 |
| `06-课程体系与补充需求判断.md` | 判断现有材料的覆盖范围与缺口 | 不替代真实牌局和用户反馈 |

正式规则优先使用公开规则原文；课程材料负责提供解释、经验和牌例；二次分析负责组织内容，不反过来成为事实来源。

## 写牌例前需要确认什么

至少记录：

1. 采用的规则名称与年份；
2. 当前级数；
3. 出牌方向；
4. 连续牌型中 A 和级牌的使用方式；
5. 炸弹、同花顺和王炸的比较方式；
6. 接风、报牌、进贡和抗贡规则；
7. 具体赛事或平台是否有补充规定。

这些条件可能直接改变牌例答案。无法确认时，应缩小结论范围，说明缺少什么条件，而不是用一个状态标签掩盖不确定性。

## 发现规则冲突时

不要直接覆盖旧结论。先记录：

- 冲突发生在哪条规则；
- 两个来源分别怎样规定；
- 来源的发布方、年份和适用场景；
- 哪些正文或牌例会受到影响；
- 本库当前采用哪个口径，以及为什么。
