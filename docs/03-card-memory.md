# 记牌与未知信息：先记真正影响判断的牌

> 本页解决：怎样在不过载的前提下，用记牌改善下一次决策。

记牌的目标不是复述全部 108 张牌，而是不断缩小未知范围，让当前决策更准确。

## 从关键牌开始

入门时可以先跟踪：

1. 大王和小王；
2. 当前级牌；
3. 与自己路线直接相关的关键点数。

等这一层稳定后，再按具体问题增加范围。例如担心顺子和同花顺时关注关键连接张；处理残局时关注对手可能形成牌型所需要的张数。

## 最基本的计算

对一类已知总量的牌，可以使用：

> 外面尚未出现的数量 = 总量 − 自己手里 − 已经公开出现

这个结果仍不告诉你牌具体在谁手里，却能排除不可能的情况。记牌真正提供的往往不是答案，而是更窄的可能范围。

## 把信息分成三层

- **确定事实**：自己手牌、已经公开打出的牌、四家公开剩余张数。
- **合理推断**：某家多次放过某牌型，可能缺少对应控制牌或在保留路线。
- **仍然未知**：未出现的牌究竟怎样分布，对手的真实意图是什么。

推断不能倒写成事实。某人一次没有接牌，可能是接不了，也可能是不愿意在这一刻消耗资源。

## 分阶段训练

### 阶段一：只记王

每一圈结束后回答：四张王已经出现几张，自己持有几张，外面还剩几张。

### 阶段二：增加级牌

在记王稳定以后，加上当前级牌和红桃级牌的出现情况。

### 阶段三：围绕一个问题记牌

例如只追踪“我的最高对子是否仍然安全”，或“对手组成五张连续牌还需要哪些关键张”。不要无目的地扩张记忆范围。

## 记牌方法怎样选择

系统课程里出现过减数计数、用手牌位置计数、指节计数和身体位置记忆等方法。它们共同解决的是“怎样持续更新数量”，但适用边界不同。

| 方法 | 适合的场景 | 本库建议 |
| --- | --- | --- |
| 减数计数 | 线下、线上、训练题 | 作为默认方法，最容易核对 |
| 纸面记录 | 单独训练和复盘 | 用于形成习惯，不带入正式牌局 |
| 手牌位置计数 | 部分线下课程 | 正式比赛要求固定持牌与出牌方式，不作为通用建议 |
| 指节或身体位置计数 | 个别课程中的辅助方法 | 可能引起动作传递信息的争议，不作为本库训练方法 |

正式规则禁止通过声响、形体或肢体动作传递非法信息。本库的训练重点是公开信息与数量更新，不训练任何可能被搭档解读为暗号的动作。

## 一道短回合练习

两副牌共有 8 张 A。假设你手里有 2 张 A，公开出牌中已经出现 4 张 A：

```text
总量：8
自己持有：2
已经出现：4
外面尚未出现：8 - 2 - 4 = 2
```

这个答案只说明外面还有 2 张 A，不能说明它们一定分布在哪一家。下一步要结合谁在对应牌型上过牌、谁仍有牌权和剩余张数继续判断。

## 常见错误

- 一开始就要求自己记住全场，导致注意力全部耗尽；
- 只记牌名，不把记录连接到当前决策；
- 把“没有看见”误写成“外面没有”；
- 用一次过牌直接判断某家没有这种牌；
- 只在复盘时觉得自己记得，实战中从不暂停核对。

## 练习模板

每局只设一个任务：

```text
本局观察对象：
初始总量：
自己持有：
已经出现：
当前未知量：
这个数字改变了哪一个选择：
最终哪里记错了：
```

当一个观察任务能够稳定完成，再增加下一类牌。范围变小不是偷懒，而是在把记忆真正转换成判断。
