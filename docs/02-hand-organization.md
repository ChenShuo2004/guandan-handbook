# 整理一手牌：先看路线，再看炸弹

> 本页解决：拿到 27 张牌后，怎样形成一条可以执行的走牌路线。
>
> 状态：方法草稿，后续补充经过复核的完整牌例。

新手整理手牌时最容易先找炸弹、同花顺和漂亮组合。但组牌真正要回答的是：整手牌大约需要几轮走完，最难处理的牌在哪里，牌权回来以后有哪些可继续的路线。

## 四个观察对象

### 1. 总手数

先粗略估算整手牌要分成多少手。减少手数通常有价值，但不是唯一目标：为了少一手而制造一个很难领出的结构，可能得不偿失。

### 2. 断点

找出最容易留到最后的散牌、弱对子或孤立结构。完整组合人人看得到，真正决定路线的常常是这些断点。

### 3. 回手能力

某个组合即使能走掉，也要问：它需要我先拿到哪一种牌权？我的大牌或炸弹能否支持这条路线？

### 4. 调整空间

哪些牌型天然完整，哪些牌可以拆分，逢人配放在哪里最有价值？保留一部分调整空间，通常比一开始把所有牌锁死更稳。

## 六步整理法

1. 确认级数，标出级牌、红桃级牌、王和明显强牌。
2. 圈出天然完整的结构，不急着拆。
3. 找出散牌和潜在断点。
4. 形成至少两种组牌方案。
5. 比较两种方案的总手数、启动条件和失败方式。
6. 给第一手写出目的：试探、清理断点、争取牌权，还是给对家留接口。

## 怎样比较两个方案

不要只比较“哪个牌型更大”，而要比较：

| 问题 | 方案 A | 方案 B |
| --- | --- | --- |
| 预计需要几手走完 |  |  |
| 最难处理的牌是什么 |  |  |
| 需要哪种领出牌型 |  |  |
| 会不会过度消耗大牌 |  |  |
| 对家可能怎样接续 |  |  |
| 失败后还有没有第二路线 |  |  |

## 一个结构例子

假设手里存在 `33 44 55 66 77` 这一段连续对子。它可以被看作三连对加两对，也可能为了配合其他牌拆成不同结构。

没有完整手牌、级数、位置和外部出牌信息时，不能断言哪种拆法一定更好。这个例子只说明：一个漂亮的局部组合，不等于整手牌的最优路线。应把它与其余断点、牌权来源和对家状态一起比较。

## 一分钟自查

拿到牌后，尝试回答：

- 我预计需要几手走完？
- 最难处理的是哪一组牌？
- 哪些结构不想轻易拆？
- 哪些牌可以作为调整资源？
- 如果我先手，第一轮准备完成什么？
- 如果计划失败，第二路线是什么？

组牌没有脱离局面的唯一答案。稳定的能力不是每次都猜中最佳组合，而是能说明每一种拆法在交换什么。
