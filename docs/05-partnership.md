# 搭档配合：从公开出牌理解对家

> 本页解决：怎样把对家纳入判断，而不把猜测当成信号。
>
> 状态：方法草稿，待补双方视角完整牌例。

掼蛋的团队目标不等于两个人各自尽快走完。局面变化时，一人可能主攻，另一人可能接应、保护牌权或阻断对手；这些角色不是固定身份，而是随手牌和剩余张数变化。

## 可以使用的信息

正常配合只能建立在公开信息上，例如：

- 对家已经打出的牌型和点数；
- 对家在哪些牌型上选择了过牌；
- 四家的公开剩余张数；
- 谁当前拥有牌权；
- 前几轮牌权怎样流转；
- 当前团队可能争取的名次组合。

身体动作、摆牌方式、约定暗号和其他隐蔽信号不属于正常配合，本库不教授也不使用。

## 不要把一次过牌解释得太满

对家没有接牌，可能意味着：

- 没有合法牌可以接；
- 有牌但成本过高；
- 希望你继续保持牌权；
- 正在保护另一条路线；
- 判断失误或没有看见。

所以，一次动作只能更新可能性，不能直接证明对家“需要什么”。需要结合后续公开动作不断修正。

## 动态角色

### 主攻

当前更有机会率先出完，团队资源应优先保护其路线。

### 接应

通过适当接牌、领出和让路，帮助主攻者继续处理手牌。

### 阻断

在对手接近出完时，优先限制其牌权和有利牌型。

一名玩家可能在几轮内从主攻转为接应。角色判断依赖当前牌面，不应在开局时永久锁定。

## 拿到牌权后的团队检查

领出以前问：

1. 对家还剩多少张，可能需要哪类入口？
2. 我准备出的牌型，对家更容易接还是对手更容易接？
3. 我是否正在压住对家的有利牌，而不是压住对手？
4. 如果对家已经形成冲刺机会，我是否需要放弃自己的局部最优？
5. 这次让路失败后，团队还有没有补救空间？

## 个人最优不等于团队最优

某个动作可能让自己的手数立刻减少，却截断对家的冲刺；另一个动作可能暂时保留自己的散牌，却让对家获得更高名次。评价选择时，应比较团队最终位置，而不是只比较自己的剩牌。

这也不意味着永远牺牲自己。若对家的路线只是弱推断，而自己的路线更确定，盲目让路同样可能损害团队。配合不是服从，而是在不完整信息下协调资源。
