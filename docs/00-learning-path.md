# 学习路径：先解决当前卡点

> 本页解决：资料很多时，应该先学什么，又怎样确认自己真的学会了。

掼蛋的入门主线可以先收成一句话：

> **规则 → 组牌 → 记牌 → 牌权与炸弹 → 搭档配合 → 残局 → 训练与复盘**

它不是唯一顺序，但比从残局口诀、高手手筋或炸弹技巧开始更稳。每一阶段都应该用一个可观察的动作结束，而不是用“文章看完了”结束。

## 第一阶段：能完整跟完一局

先掌握四件事：认出常见牌型，判断自己能否接牌，知道什么时候轮到自己，遇到争议时知道应核对规则版本。

学习材料：[基本规则](01-rules.md)

完成标志：不依赖旁人逐手提醒，也能跟完一局。

## 第二阶段：能说出整手牌的路线

拿到 27 张牌后，不只找炸弹。先估算要分几手走完，找出最难处理的散牌，并说明哪些结构稳定、哪些牌可以调整。

学习材料：[整理一手牌](02-hand-organization.md)

完成标志：一分钟内能说出预计手数、主要断点和第一手的目的。

## 第三阶段：用记牌缩小未知

先跟踪王、当前级牌等真正影响判断的牌。每局只设一个观察任务，稳定以后再增加范围。

学习材料：[记牌与未知信息](03-card-memory.md)

完成标志：能够用“总量 − 自己手里 − 已经出现”核对关键牌余量，而不是凭模糊印象猜。

## 第四阶段：压牌前先想后续

从“我能不能压”转向“压完以后怎样”。大牌和炸弹的价值取决于它们换来了什么，以及拿到牌权后有没有下一步。

学习材料：[牌权与炸弹](04-control-and-bombs.md)

完成标志：出大牌前能补完一句话：

> 我付出这张牌，是为了 ______；拿到牌权以后，我准备 ______。

## 第五阶段：把对家放进决策

观察对家的公开出牌、剩余张数和可能角色。不要只问自己怎样走完，也要检查自己是否截断了对家的路线。

学习材料：[搭档配合](05-partnership.md)

完成标志：复盘时能分别说明自己与对家当时可能承担的角色，以及牌权怎样在团队内流转。

## 第六阶段：让残局推断可以检查

报牌数字只会缩小可能性，不会直接给出答案。把信息分成确定事实、基于事实的推断和仍然未知的部分。

学习材料：[残局判断](06-endgame.md)

完成标志：不再只说“他大概是一手牌”，而能说明依据、替代可能和猜错后的最大损失。

## 第七阶段：形成训练闭环

教程负责帮助理解，训练负责形成能力。最小闭环是：

> 看到场景 → 先作选择 → 获得解释 → 隔一段时间再做变式

学习材料：[训练与复盘](07-training.md)

完成标志：能记录自己的错因，并在相似但不相同的场景里修正判断。

## 资料怎么筛

面对一篇教程、视频或复盘，可以先问：

1. 来源和规则版本是什么？
2. 它适合什么水平的人？
3. 牌例有没有交代位置、剩余张数和牌权？
4. 它有没有解释其他选择为什么较差？
5. 结论属于规则、经验、概率判断，还是个人观点？

回答不出这些问题的材料可以提供灵感，但不适合作为学习主线。
