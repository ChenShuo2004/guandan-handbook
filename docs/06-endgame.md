# 残局判断：剩余张数只会缩小可能性

> 本页解决：报牌以后，怎样利用有限信息作出可检查的判断。
>
> 状态：方法草稿，待补多解残局题。

对手剩 5 张，不等于他一定是一手顺子。它也可能是三带对、五张炸弹，或几组没有连起来的牌。剩余张数提供约束，却不会自动给出牌型。

## 残局需要的完整条件

分析一个残局时，至少记录：

- 当前级数和规则版本；
- 四家的位置与剩余张数；
- 当前轮到谁，牌权在谁手里；
- 最近几轮的出牌顺序；
- 已经确认出现的关键牌；
- 自己的完整手牌；
- 当前团队目标。

缺少这些条件时，结论应该相应减弱。

## 四步判断法

### 1. 写下确定事实

只记录可以直接核对的信息，不加入动机解释。

### 2. 列出仍然可能的牌型

用剩余张数、已见牌和规则排除不可能情况。保留至少两个仍能解释公开信息的方案。

### 3. 比较失败代价

不是只问“哪个最可能”，还要问“如果猜错，哪一种会直接结束牌局或破坏团队名次”。概率略低但代价极大的路线，可能值得防范。

### 4. 选择保留补救空间的动作

在多个方案都无法确认时，优先比较哪些动作能覆盖更多可能性，或能在下一轮继续获得信息。

## 事实、推断、假设

```text
事实：下家公开报剩 5 张，本轮仍未领出。
推断：他可能保留一手五张牌型，也可能是两手牌。
假设：他手里是一条顺子。
风险：如果把假设当事实而领出连接牌，可能帮助他直接走完。
```

这套写法的目的不是消除猜测，而是让猜测有边界。

## 残局题应该怎样写

一份可复用的残局题不应只给“标准答案”，还应包含：

1. 完整公开条件；
2. 至少两个可选动作；
3. 每个动作依赖的假设；
4. 对手不同持牌下的失败方式；
5. 为什么最终方案在当前目标下更稳；
6. 哪些规则变化会改变答案。

## 常见错误

- 把报牌数字直接等同于某种牌型；
- 只记住自己担心的可能，忽略其他解释；
- 追求猜中率，不比较猜错的损失；
- 看到最终底牌后，用结果倒推当时只有一个正确选择；
- 牌例条件不完整，却使用绝对语气。

## 自查清单

- 我确定知道什么？
- 我根据什么作出推断？
- 还有哪些情况能解释同样的公开动作？
- 哪一种猜错后的损失最大？
- 有没有一个动作能保留更多补救空间？
