# 牌权与炸弹：压住以后准备做什么

> 本页解决：什么时候值得争牌权，以及炸弹真正改变了什么。

“能压”只说明动作合法，不说明动作值得。大牌和炸弹的价值来自它们改变了牌权、出牌顺序和团队路线，而不是来自牌面本身足够醒目。

## 牌权是什么

牌权是下一轮的领出权。它让你能够重新选择牌型，因此可能用来：

- 清理自己的断点；
- 启动一个只有领出时才容易走掉的结构；
- 给对家送出更容易承接的牌型；
- 阻止对手在剩牌很少时继续领出；
- 调整团队内的主攻与接应角色。

如果拿回牌权后没有清楚的下一手，刚才的压制可能只是在消耗资源。

## 出大牌前的两个空

每次准备使用控制牌或炸弹前，补完：

> 我现在付出这张牌，是为了 ______；拿到牌权以后，我准备 ______。

第一个空说明目标，第二个空说明后续。第二个空填不出来时，至少应该比较一次过牌的结果。

## 比较“压”和“不压”

可以按以下顺序检查：

1. 当前领出者是谁，对家是否仍可能接手？
2. 如果我不压，最可能发生什么？
3. 如果我压，必须付出什么资源？
4. 拿到牌权后，我能启动哪条路线？
5. 这条路线被反制后，还有没有补救？
6. 这次行动改善的是个人手牌，还是团队结果？两者是否冲突？

还可以把判断压缩成四个变量：

| 变量 | 要回答的问题 |
| --- | --- |
| 角色 | 我现在更像主攻、接应还是阻断？ |
| 阶段 | 牌局处于开局、中盘还是有人即将冲刺的残局？ |
| 目标 | 当前要争上游、保对家，还是先避免双下？ |
| 后续 | 取得牌权后，下一手具体领出什么？ |

课程材料里常见“主攻要抢牌权”“助攻要让牌”等说法。它们只有在角色、阶段和后续路线同时明确时才有意义，不能独立变成口诀。

## 炸弹的三个作用

炸弹通常可能承担三种任务：

- **止损**：阻断即将出完的对手；
- **转换**：夺回牌权，切换到本方更有利的牌型；
- **保护**：为自己或对家的关键路线提供一次确定性更高的回手。

“炸弹越晚用越好”和“有机会就炸”都是缺少条件的口诀。应比较炸前与炸后的局面，而不是只比较舍不舍得。

## 一个最小场景

假设下家只剩一手牌，并且正在领出；你有炸弹可以阻断。是否使用，至少还取决于：

- 你是否确定那是一手可直接走完的合法牌型；
- 对家有没有更低成本的阻断方式；
- 炸完以后你准备领出什么；
- 你的炸弹是否还承担另一个更关键的保护任务；
- 当前团队目标是争上游、保住二游，还是避免双下。

仅凭“对手剩得少”不能推出唯一动作，但剩牌少会提高不行动的风险。好的分析应把风险变化写出来。

## 炸弹不是库存，而是一次转换

可以用一张简短的炸前／炸后表检查：

| 检查项 | 炸之前 | 炸之后 |
| --- | --- | --- |
| 牌权属于谁 |  |  |
| 最接近出完的是谁 |  |  |
| 自己剩余难处理结构 |  |  |
| 对家可接续牌型 |  |  |
| 对手最危险的路线 |  |  |
| 本方还剩几次可靠回手 |  |  |

如果炸完只改变了第一行，却没有改善后面几行，这次炸弹的收益可能没有想象中大。

## 复盘问题

- 我争到的每一次牌权，下一手做了什么？
- 哪次大牌只是赢了一手，没有改善路线？
- 哪次过牌给对家留下了更好的接续？
- 炸弹使用后，四家的剩余手数怎样变化？
- 如果结果相反，我仍会认为当时的判断合理吗？
